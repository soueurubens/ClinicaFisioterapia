# Clínica de Fisioterapia

## Front-end

## Página Inicial( Index )
- Essa página conterá um card com a seguinte pergunta: "Você é paciente ou funcionário?"
- Caso o usuário clique em paciente ele será levado para a página do paciente
- Caso o usuário clique em funcionário ele será levado para a página Login

## Página Login
- Essa página conterá um campo de preenchimento com nome de usuário e senha
- Depois de digitar as credenciais corretas e clicar em entrar o usuário será levado para a página Dashboard

## Página Dashboard
- Essa página conterá uma barra de navegação com as opções "Agendamentos", "Prontuários", "Prescrições" e "Sair"
- Ao clicar em sair o usuário voltará para a página Inicial
- Quanto as demais, ele será redirecionado de acordo com o que ele escolher
- Além da barra de navegação, terá um card mostrando as consultas do dia

## Página Agendamentos
- Essa página conterá todos os agendamentos(em formato de calendário)
- Usuário poderá narcar e cancelar consultas
- Usuário poderá ver e editar o status da consulta(confirmada, em andamento, finalizada)

## Prontuário
- Essa página conterá o histórico de pacientes
- Barra de procura por nome do paciente
- Ao clicar no nome da lista abrirá um modal com informações adicionais
- Essas informações poderão ser editadas
- A cada paciente poderão ser atrelados arquivos(ficha cadastral, histórico médico, exame físico, plano de tratamento, evolução)
